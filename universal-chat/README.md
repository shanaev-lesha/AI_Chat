# Universal AI Chat

Небольшой pet-проект — экспериментальный AI-чат, в котором собраны несколько популярных подходов к построению AI-ассистентов.

Цель проекта — попробовать разные архитектуры (RAG, multi-agent системы, голосовые ответы) и объединить их в одном простом backend.

## Возможности

### LLM Chat

Обычный чат с языковой моделью через OpenRouter.

Можно выбирать разные модели для ответа.

### RAG (Retrieval Augmented Generation)

Поддерживается загрузка PDF и ответы на вопросы по содержимому документа.

Pipeline:

PDF → chunking → поиск релевантного текста → LLM ответ.

### PDF Summarization

Можно загрузить большой PDF и получить краткое структурированное summary.

Используется chunking + финальная агрегация summary.

### Multi-Agent Debate

Два агента обсуждают тему:

Agent A — критическая позиция
Agent B — оптимистичная позиция

После дебатов третий агент (Judge) оценивает аргументы и выбирает победителя.

### Self-Improving Agents

После раунда дебатов агенты анализируют feedback судьи и улучшают свои промпты перед следующим раундом.

### Text-to-Speech

Ответы можно озвучивать через Deepgram TTS.

## Стек

* Node.js
* Express
* OpenRouter API
* Deepgram
* pdf-parse

## Основные API

POST /chat — чат с LLM
POST /upload — загрузка PDF
POST /summarize — суммаризация PDF
POST /agents — мульти-агентная дискуссия
POST /voice — генерация голоса
POST /reset — очистка истории

## Цель проекта

Это экспериментальный проект для изучения:

* RAG архитектуры
* multi-agent взаимодействия
* prompt engineering
* интеграции голосовых интерфейсов

Проект сделан максимально простым, чтобы можно было быстро тестировать новые идеи.
